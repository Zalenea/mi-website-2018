Wie funktioniert Kommunikation in der Medieninformatik?

Auf dieser kleinen Website werden einige Grundsätze und Ausprägungen bezogen auf die Kommunikation des Lehr- und Forschungsgebiets Medieninformatik am Campus Gummersbach dargelegt. Dies ist eine offener, iterativer Prozess, an dem sich Studierende, Mitarbeiter und Ehemalige gerne beteiligen können. Für Ideen und Fragen bitte die GitHub Issues des zugehörigen GitHub Repos benutzen.
